## Development

### Public functions

- Are always `async`
- First argument is always the state

### Setup

```bash
yarn install
```

### Tests

```bash
yarn test
```

### Theme

Use the [default theme](https://github.com/dennisreimann/uiengine-theme-react) as a basis for your customizations.
